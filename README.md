# CodeAlpha: Unemployment Analysis with Python

This repository contains my submission for the CodeAlpha Data Science Internship task.

## Project Overview
The goal of this project is to analyze unemployment rate data representing unemployed people percentages, investigate the impact of Covid-19 on unemployment rates, and identify key patterns or seasonal trends.

## Tech Stack
* Python
* Pandas
* Matplotlib

## Results
The analysis successfully isolated the sharp spike during the Covid-19 pandemic and mapped key seasonal employment trends across regions.
